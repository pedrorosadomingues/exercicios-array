Verificando disjuntores
se uma caixa de disjuntores está confirmada. Su que o item está indicando um array onde está sendo identificado o disjuntor (começa) eo valor do item está ligado ou boolean se o disjuntor ou está ligado ou disjuntor. imprima no console apenas os disjuntores que estão ligados. Se possível, utilizar o conceito de veracidade.

Exemplo: Tendo a matriz de 8 disjuntores a seguir:

const disjuntores = [false, false, true, false, false, true, false, false];

// seu codigo aqui
Escreva o código que imprimirá no console:

2
5
