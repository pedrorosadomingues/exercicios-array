Filtrando números
Declare um array contendo alguns números quaisquer.

Depois choram um novo array0 e maiores apenas os números do array original que são 100 ou que são 100 finais, que guardam o novo array.

Exemplo:

Para o array original abaixo:

const original = [5, 7, 13, 17, 26, 34, 118, 245];

// seu codigo aqui
Deverá ser impresso no console:

[13, 17, 118, 245]
